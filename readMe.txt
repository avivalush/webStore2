ברוכים הבאים לצד השרת בחנות התקליטים שלנו:
לפני שנתחיל ישנם מספר דברים עליכם אותם לדעת!

* הפורט שעליו צריך להתחבר הוא - 3000 *
* קישור לגיט:  https://github.com/avivalush/webStore2  *

הערות נוספות:
1. בפונקציית ה Login - התאריך הוא אינו פרמטר שמוזן על ידי הלקוח, אלא תאריך שנדלה בחלק הcllient על מנת לשמור על ההתחברות האחרונה.
2. בפונקציית ה Register - בעת יצירת משתמש חדש, ניקח את שדה המדינה מקובץ ה XML בצד הclient. כמו כן, שאלות האחזור יהיו - "שם האם" ו"שם בית הספר היסודי".
3. בפונקציית ה Buy - נבדוק בצד הלקוח האם המוצר קיים במלאי ורק אז נבצע את הרכישה.
4. שינוי מטבע הקנייה יתבצע בצד הלקוח.
5. הטמעת הcoockie אצל המשתמשים תבוצע בצד הלקוח
6. הגבלות קלט של שם משתמש וסיסמת המשתמש יתבצעו בצד הלקוח
